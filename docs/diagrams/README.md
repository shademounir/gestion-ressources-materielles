## Convention proposee

Les fichiers de diagrammes pourront etre organises par type :

```text
docs/diagrams/
├── bpmn/
├── use-cases/
├── uml/
└── architecture/
```

Cette structure sera creee uniquement lorsque les premiers diagrammes seront produits.
